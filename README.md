## Requirements
- Python 3.8+
- git

## Quickstart

Clone and install dependencies:
```bash
git clone https://github.com/mannansainicyber/ZeroTrust.git
cd ZeroTrust
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Running

Run the application:
```bash
python app.py
```
